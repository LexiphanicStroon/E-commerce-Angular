E commerce app made with ASP.NET Core and Angular. Methods for adding to cart, calculating total item cost, and adding to cart. The cart persists upon page refresh, so what was in your cart will still be there. You can search by keyword, filter by brand and type, and reset filters. Includes HTTP error handling.
